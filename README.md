# Customer List

Customer list of Flozka, local florist in Batam

## Features

1. Show Contacts
2. Add Contact
3. Edit Contact
4. Delete Contact

## Flowchart

```mermaid
  graph TD;
    A[Start]-->B[Show All Contacts];
    B-->C{Get User Choice };
    C-->|One| D[Add Contact];
    C-->|Two| E[Edit Contact];
    C-->|Three| F[Delete Contact];
```
